PROYEK AKADEMIS

Lokasi UMKM

Melakukan penelitian di sebuah UMKM parfum dengan tujuan untuk membantu usaha tersebut dalam memahami performa penjualannya dengan lebih baik melalui visualisasi data KPI. Dengan mengolah data transaksi yang ada, saya menyajikan informasi seperti tren penjualan bulanan, parfum terlaris, serta kontribusi tiap produk dalam bentuk grafik yang mudah dipahami, sehingga dapat mendukung pengambilan keputusan bisnis secara lebih tepat.

Desain Database

Mengumpulkan data parfum dan data transaksi dari toko tersebut. Data tersebut kemudian dibersihkan (data cleaning) untuk menghilangkan duplikasi dan memperbaiki format. Setelah itu, saya mendesain database menggunakan MySQL untuk mengelola data dengan rapi. Selanjutnya, data divisualisasikan menggunakan PHP dan Chart.js dalam bentuk grafik KPI seperti tren penjualan, parfum terlaris, dan capaian target untuk membantu pemilik toko memahami performa penjualan secara efektif.
